T32cmm_4_crane is a simple tool for analysing memory dump of Crane. Please make sure T32 simulator installed. 
  
1.Change T32PATH in _t32cmm_4_crane.bat according with local path.
2.Run _t32cmm_4_crane.bat. Or run t32.cmm by File->Run Script from TRACE32 simulator.
3.Choose *.axf.
4.Choose com_DDR_RW.bin.	//It is recommanded to save all the com*.bin to one folder. especially com_DTCM.BIN com_ITCM.BIN com_dsp_ddr.BIN
5.There will be a menu named "ThreadX" if T32PATH exist.
